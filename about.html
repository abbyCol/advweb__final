About
Welcome to this Lord of the Rings fansite, in which characters and locations are explored with brief descriptions intended to make any person familiar with different lore concepts.

All images from Adobe Stock images as the original Lord of the Rings images were unable to be used. Popup background utilized from rawpixel. Animations created with CSS animations. Overlay buttons created by Niall in which it was changed to match the theme of this website and integrated to function. Video linked to Troll image sourced from YouTube.
